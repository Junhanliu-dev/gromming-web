Suggestion of setting up server

1.Download xampp with the link : https://www.apachefriends.org/zh_cn/index.html

2.After installing of xampp, select apache

3.Paste the directory of “triganle_sprint1” into htdocs under the xampp installed directory

4.Open browser

5.Enter url “127.0.0.1/‘path/index.html’”
